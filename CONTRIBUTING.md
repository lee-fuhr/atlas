# Contributing

The Atlas framework is open to contributions:

- **Typos and clarity fixes** — PRs welcome, no issue needed
- **Improvements to the KCA model explanation** — open an issue describing what's unclear
- **New examples for placement tests or arrows** — PRs welcome with concrete scenarios
- **Corrections to the applied example** — open an issue if categorization seems wrong

The evolution protocol (section 11 of the framework) describes how structural changes are evaluated.
